# pixiv-dl-nodejs
Pixivダウンローダーです。

https://www.pixiv.net/

**自己責任**でご利用ください。

**個人使用**を意図して作成されています。

お住まいの管轄地域の法律に従ってください。


# 依存関係のインストール
これを実行するだけ：
`npm install node-fetch adm-zip fluent-ffmpeg selenium-webdriver`
また、ffmpegのインストールが別途必要です。

# テスト済み環境
**nodejs** v23.3.0

**npm** 10.9.0
