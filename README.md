# pixiv-dl-nodejs
Pixiv Downloader in Nodejs

https://pixiv.net/

# Install 
`npm install node-fetch adm-zip fluent-ffmpeg`

Just it.

# My env
```
$ node -v
v23.3.0
$ npm -v
10.9.0
```
